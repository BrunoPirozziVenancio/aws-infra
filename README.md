# aws-infra

Projetos de infraestrutura na AWS usando Terraform e Ansible.

## Projetos Terraform

- **aws-alb-ec2** — Load Balancer + EC2
- **aws-ec2-userdata** — Instância EC2 com User Data
- **aws-s3-upload** — Bucket S3 + Script Bash de Upload
- **aws-lambda-monitoring** — Lambda para monitoramento

## Ansible

- Playbooks para provisionamento e configuração de instâncias AWS
