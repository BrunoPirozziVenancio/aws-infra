# Exemplo: VPC + EC2 + S3
Exemplo pratico de uso dos modulos vpc, ec2 e s3 em conjunto.
## O que este exemplo cria
- VPC com 2 subnets publicas em AZs diferentes
- Instancia EC2 t2.micro na primeira subnet
- Bucket S3 com versionamento e criptografia
## Como usar
terraform init
terraform plan
terraform apply
terraform destroy
